# Day4 Spring Boot Project

Project Spring Boot Day4 sử dụng các thư viện:
- Spring Data JPA
- Spring Web
- Lombok
- MySQL Connector

## Khởi động project

1. Cấu hình database trong `src/main/resources/application.properties` hoặc `application.yml`.
2. Chạy lệnh sau để build và khởi động:

```
mvn clean install
mvn spring-boot:run
```
